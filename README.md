# Bank Management System

A console-based banking application developed using Java.

## Features

- Create Account
- Deposit Money
- Withdraw Money
- Transfer Money
- View Transaction History
- File Persistence

## Technologies

- Java
- OOP
- Collections
- File Handling
- Exception Handling
- Maven

## Project Structure

src
├── model
├── service
├── repository
├── ui

## How to Run

git clone ...

mvn clean install

java Main
